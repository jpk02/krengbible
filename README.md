# Infrastructure Daily Digest — Archive

Durable archive of the daily infrastructure news digest (US-focused, with major global deals and projects flagged).  Covers oil & gas, power & renewables, data centers, aviation, rail, ports, and infrastructure finance.

Generated automatically each morning at 08:00 UTC (4:00 AM ET) by a scheduled Claude Code job.  The same digest is emailed on delivery; this archive is the searchable, dated record.

## Layout

- `digests/YYYY-MM-DD.md` — one file per day.

## Note

This lives on the `infra-digest` orphan branch of the `krengbible` repo.  It shares no history or content with `main`.  It can be lifted into its own standalone repo at any time via `git subtree split` (history preserved) or by copying the `digests/` folder (files are self-contained).
